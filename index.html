# 前端工程化

  • 工程化是系统化，模块化，规范化的过程
  • 能够提高整个系统生产效率

# 前端工程化利器webpack

  1. 编译，JavaScript，css等编译
  2. 文件的压缩，打包，合并，公共模块提取等
  3. 图片等资源的处理
  4. webpack-dev-server,eslint,热更新等辅助开发的工具
  5. tree-shaking等优化js工具

# 安装

全局安装 （版本大于4需要再安装webpack-cli）

npm install -g webpack (npm install -g webpack-cli)

# 项目结构

├── node_modules                                // 包管理工具下载的文件
├── src                                         // 编写的js等文件
├── index.html                                  // 入口文件html
├── app.js                                      // 入口js文件
├── webpack.config.js                           // webpack配置文件
├── package.json
