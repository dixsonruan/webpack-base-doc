# 四个基本配置

entry, output, loader, plugin是webpack四个既基本又核心的配置

在webpack.config.js中编写

# entry

entry是webpack的打包入口，是程序开始的起点，代码将从这里编译

module.exports = {
    entry: string|Array<string>
}

在entry中填写入口文件名称，webpack会自动找到填写的文件，将文件中引用的依赖自动加载

# output

output是webpack的打包出口，最终的打包结果根据output的定义输出

module.exports = {
    output: {
        filename: string //指定的出口文件名称
    }
}

当填写完这两个配置后，在当前文件夹下打开终端，输入命令webpack，便可开始一次打包
